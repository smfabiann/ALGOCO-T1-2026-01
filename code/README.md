## Multiplicación de matrices

### Programa principal
La forma que esta hecha es que iterará por una lista con los nombres de los algoritmos. Por cada algoritmo
obtendra la hora inicial y la final usando high precision clock, asi, obteniendo una medicion en ms.
Estas mediciones seran guardadas en un archivo .csv. Cada vez que un algoritmo termina su ejecucion, se creara
o se agregara al archivo txt llamado "completion_list.txt" (ubicado en data/measurements/) el nombre del
algoritmo, esto para que no se vuelva a ejecutar el algoritmo cuando ya se obtuvieron los datos.

### Scripts
Se uso pandas y matplot para obtener los graficos en funcion de Tiempo (ms) vs N (tamaño entrada) y Delta de Memoria.
